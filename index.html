<div>

The <div> tag defines a division or a section in an HTML document. The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The <div> tag is easily styled by using the class or id attribute. Any sort of content can be put inside the <div> tag! It's even possible to make a html document consisting of only <div> tags.

<span>

The <span> tag is an inline container used to mark up a part of a text, or a part of a document. The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The <span> tag is much like the <div> element, but <div> is a block-level element and <span> is an inline element.

<a>

The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination. By default, links will appear as follows in all browsers: An unvisited link is underlined and blue; A visited link is underlined and purple; An active link is underlined and red.

CSS tricks


mix-blend-mode

The mix-blend-mode property specifies how an element's content should blend with its direct parent background.

Example: The same image with it's mix-blend-mode property set to normal and darken.

clip-path

The clip-path property in CSS allows you to specify a specific region of an element to display, with the rest being hidden (or “clipped”) away.

Example: An image with no clip path, and the same image with it's clip-path set to ellipse(45% 45% at 50% 50%).

:hover Selector

The :hover selector is used to select elements when you mouse over them. While it is most commonly used on links, it can be used on any element.

Example: Hover over the sleeping possum to wake him up!
